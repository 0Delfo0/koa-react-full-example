[![Build Status][travis.img]][travis.url]
[![dependencies][deps.img]][deps.url]

Full example using Koa, React, Mongoose, Gulp, Mocha, and on Travis

### Build

`gulp` or `gulp install`

### Develop

`npm start` or `gulp dev`

### Test

`npm test`
